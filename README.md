# PSUTools

## Requirements
* [Lua 5.1](https://www.lua.org/)
* [Node v14.15.4](https://nodejs.org/en/)

## Usage
```bash
npm start
```

## Features

### Grab Premium
Grabs the original obfuscated script behind PSU premium output.

### Constant Dump
Grabs the constants.
