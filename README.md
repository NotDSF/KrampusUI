# PSUTools

## Requirements
* [Lua 5.1](https://www.lua.org/)
* [Node v14.15.4](https://nodejs.org/en/)

## Usage & Installation

### Installation
```bash
npm i
```

### Usage
```bash
npm start
```

## Features

### Grab Premium
Grabs the obfuscated script behind premium output. This bypasses their low attempt at security.

### Constant Dump
Grabs the constant values.
